# Edu-Navigator
It is a career counselling project majorly for the students so that they can get enrolled into a right track. This project is created for a hackathon named **SemiColon**.
The team members are Suvam Bista [github](https://github.com/sandybeast07), Padam Prasad Dhakal [github](https://github.com/PadamPrasadDhakal), Yugal Niroula [github](https://github.com/YugalX49). The frontend is covered by Suvam Bista, backend by Padam Prasad Dhakal and research and designing is covered by Yugal Niroula.
